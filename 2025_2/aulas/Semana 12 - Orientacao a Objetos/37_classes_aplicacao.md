# Orientação a Objetos: Aplicação

## Base de dados

Você recebeu um conjunto de dados contendo diversos jogos.
Esse conjunto de dados possui atributos variados sobre cada jogo, como por exemplo:
* título;
* preços (original e com desconto);
* data de lançamento;
* resumo das avaliações recentes;
* resumo das avaliações gerais;
* número de reviews;
* linguagens suportadas;
* tags populares;
* ..

## Tarefa

> Projete e implemente uma solução orientada a objetos que permita:
>
> 1. aplicar múltiplas formas de filtrar a lista de jogos
> 2. escolher diferentes formas de ordenar os jogos resultantes

### Requisitos mínimos

* a solução deve permitir a criação de **novos tipos de filtros** sem alterar o código existente
* a solução deve permitir a criação de **novas formas de ordenação** sem alterar o código existente
* devem existir **pelo menos 3 filtros diferentes**
* devem existir **pelo menos 2 formas de ordenação diferentes**

### Critério de sucesso

Dada uma lista de jogos, deve ser possível combinar qualquer conjunto de filtros com qualquer forma de ordenação para obter uma lista final ranqueada.
Você decide como projetar isso. O design é parte do exercício.
